All verilog files are located in the appropriate directories:
blue/ - system level modules, memory map, misc. IO
blue/alu/ - alu modules
blue/gpu/ - graphics modules
blue/sound/ - sound modules

Other directories:
blue/asm/ - assembler
blue/blue_project - ISE project. All Verilog files are symbolically linked
blue/docs - Documentation and datasheets
blue/gpu/utils - Scripts for converting png images into sprite table dat files and back. 

Duck hunt:
blue/duckhunt - pseudo code, assembly code
blue/duckhunt/images - sprite tables, palette files
